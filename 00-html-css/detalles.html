Software Developer Senior

Tech Solutions Inc. | Remoto Full Stack

Descripción del puesto

Tech Solutions Inc. está buscando un Ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. EI candidato ideal tendrá una sólida formación en desarrollo de software, con experiencia en eI diseño, desarrollo e implementación de soluciones de software escalables y de alto rendimiento. Como Ingeniero de Software Senior, usted será responsable de liderar proyectos de desarrollo, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos de software de alta calidad.

Responsabilidades

  • Diseñar y desarrollar soluciones de software escalables y de alto rendimiento.
  • Colaborar con equipos multifuncionales para entregar productos de software de alta calidad.
  • Liderar proyectos de desarrollo, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos de software de alta calidad.
  • Participar en la planificación y ejecución de pruebas de software.
  • Realizar revisiones de codigo y proporcionar retroalimentacion constructiva a los miembros del equipo

Requisitos

  • Licenciatura en informática o campo relacionado.
  • Mínimo de 5 años de experiencia en desarrollo de software
  • Experiencia con frameworks de JavaScript (por ejemplo, React. Angular, Vue.js)
  • Familiaridad con metodologías ágiles y herramientas de control de versiones (Git)

Acerca de la empresa

Tech Solutions Inc. es una empresa global de soluciones tecnológicas que ofrece servicios de desarrollo de software, consultoría y asesoría a clientes en todo el mundo. Nuestro equipo de profesionales altamente calificados está disponible para ayudarle a crear soluciones de software de alta calidad y eficientes.
